操作模式 描述
‘r’ 默认模式，即只读模式
‘w’ 写入模式。如果文件不存在，则创建一个文件
‘x’ 创建新文件，如果文件存在，则操作失败
‘a’ 追加模式，如果文件不存在，则创建一个文件
‘b’ 以二进制模式打开文件
‘+’ 读写模式， 适合更新文件
